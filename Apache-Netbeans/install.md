# install Apache Netbeans

## on Windows

+ [download Apache Netbeans](https://netbeans.apache.org/download/)

## Mac os

+ [download Apache Netbeans](https://netbeans.apache.org/download/)

## on Linux with snapcraft

+ Ubuntu

install snapcraft

    ```sh
    sudo apt instal snapd
    ```

install Apache Netbeans using snap

    ```sh
    sudo snap install Netbeans --classic
    ```
